# TimeSeriesAnalysis
Time Series Analysis

Набор данных: Использую для работы над этим проектом набор данных временных рядов из супермаркета Kaggle(https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting?select=train.csv). В наборе данных убрала столбцы,которые не нужны мне при анализе. По итогу:набор данных содержит 8 столбцов, а именно: идентификатор строки, идентификатор заказа, дату заказа, дату отгрузки, режим отгрузки, идентификатор клиента, идентификатор товара и продажи.

И выполнила следущие задачи:
1. Используйте функцию окна lead() для создания нового столбца sales_next, который отображает продажи в следующей строке набора данных.

2. Создайте новый столбец sales_previous для отображения значений строки над заданной строкой.

3. Ранжируйте данные на основе продаж в порядке убывания, используя функцию RANK().

4.Используйте обычные команды SQL и агрегированные функции для отображения средних показателей продаж за месяц и день.
5.Оценивайте скользящие средние с помощью оконных функций.
